## <span id = "anchorPoint">**锚点**</span>
> ### 创建锚点
>> ```string 
>> 方式一：
>> [说明文字](#jump)
>> 
>> 方式二(html方式)：
>> <a href="#jump" target="_self">说明文字</a>
>> ``` 

> ### 标记跳转位置
>> ```string 
>> <span id = "jump">跳转到的位置</span>
>> ```
